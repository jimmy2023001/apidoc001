# 核心组件

核心组件，目前大部分放在：/Api/Core 目录下，这些组件贯穿整个请求生命周期

通过引入插件：Plugin, 可以很方便的调度这些系统内置服务

## Loader [自动加载]

## Router [请求解析调度]

## Request  [请求解析]

## Response [响应调度]

## Browser [UA识别]

## Crypt  [加解密套件]

## Db     [数据库操作]

## Redis  [缓存服务]

## Plugin [通用插件]

## Session [授权服务：TOKEN]

## Cookie [PC端客户识别]

## Permission [访问综合检查]
